== Description ==

1- How to use the plugin ? 

+ Install this plugin.
+ Install the following chrome extension to be able to use the plugin <a href="https://goo.gl/yN7Cns">install extension</a>
Please Note: if you cannot unstall the chrome extension, check the FAQ section. 


2- Features

Are you a reseller of Bigbuy products and you�re tired of copying each product one at a time, to sell?
The bigbuy woocommerce importer plugin is the Solution you�ve been looking for!

Bigbuy woocommerce importer is a plugin that offers you all the tools you need to import and sell your products from Bigbuy webpage to your Woocommerce shop effortlessly. 

Bigbuy woocommerce importer App enables you to customize products before you import them. With the App, you can;

Customize and import product short descriptio, title, price
Customize and import product Variations.
Customize and import product description usign an advanced editor.
Customize and create Product Reviews and rating.
Customize create and import product specifications.
Customize and import product weight.
Customize and import product sku, sale price and regular price for variable products. 
Edit images using advanced image editor tool.
Set price formula to automate price calculations (multi intervall).
Select the product categories you wish to import the products to.
Import images to your media library from the Original product images.
Import images to your media library from the product description.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. install the extension chrome. <a href="https://goo.gl/yN7Cns">install app from here</a>
3. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= I cannot install the extension, I see Price not available  =

your country does not allow international payments for google chrome extensions, you must install 

a vpn, we recommend the following chrome extension https://goo.gl/JViVc2  once you install it, click on the 

extension icon on top of the browser and set you country to united stated. reload the extension page.

yes for sure, you can use another laptop, you only need to access with the google account you have used to purchase the extension.
